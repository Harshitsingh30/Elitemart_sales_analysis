Project Overview 

1. Project Title: 
EliteMart Sales Analytics Dashboard using Excel

3. Objective: 
To analyze sales performance, order patterns, and profit/loss trends, and to provide actionable 
business insights through an interactive Excel dashboard.

4. Dataset Source: 
The dataset used for this project is sourced from Kaggle.
 
5. Tools Used: 
● Microsoft Excel 
● Power Query Editor 
● Pivot Tables 
● Charts 
● Slicers

6. Steps Performed:
Data Cleaning (Using Power Query) 
● Removed duplicate records 
● Handled missing values 
● Fixed data type errors 
● Standardized date formats
 
Data Preparation 
● Extracted Month and Year from order date 
● Created structured dataset for analysis 
● Organized columns for better usability 

Pivot Table Creation 
● Orders by State 
● Orders by Region 
● Orders by Category 
● Profit by Category 
● Sales by Segment 
● Loss by Product Type 

Dashboard Development 
● Created KPI cards (Orders, Revenue,Top ordered product_type, Profit, Loss) 
● Designed charts (Bar, Column, Pie) 
● Added slicers for Year and Month 
● Built an interactive and user-friendly dashboard 

7. Key Insights: 
● The West region generates the highest number of orders and overall sales. 
● The South region records the lowest sales performance. 
● Certain product types contribute significantly to losses, especially when higher 
discounts are applied. 
● The Furniture category shows lower profitability compared to other categories. 
● The Consumer segment contributes the highest number of orders.
● Sales and orders peak during September to December, indicating seasonal demand. 

11. Recommendations: 
● Reduce excessive discounts on loss-making product types.
● Improve marketing strategies in the South region to increase sales. 
● Maintain sufficient inventory from September to December to meet peak demand. 
● Focus on high-performing categories and optimize low-profit segments like Furniture.

8. Conclusion:
This project demonstrates how raw sales data can be transformed into meaningful insights 
using Excel. The dashboard enables better decision-making by highlighting key trends, problem 
areas, and opportunities for business growth.
